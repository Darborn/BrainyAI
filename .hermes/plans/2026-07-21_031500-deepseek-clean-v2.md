# 全部迁移至 API Key 模式 v2 — 实施计划

> **Goal:** 删除所有"偷 token"的旧 provider，保留全部用户功能（对话/摘要/文件上传/划词），以 DeepSeek API 重新实现后端。

**关键原则:** 删的只是 provider 层（怎么调 AI），不删功能层（用户能做什么）。

## Phase 0: 大扫除

按上面「删除」列表删除所有文件，每个步骤后运行 `tsc --noEmit` 确保不破坏保留代码。

### Step 0.1: 删 libs/ 旧 provider

```bash
rm -rf libs/chatbot/openai libs/chatbot/perplexity libs/chatbot/copilot libs/chatbot/kimi
rm -rf libs/open-ai libs/copilot
rm libs/ga.ts
```

### Step 0.2: 删 contents/ 旧脚本（保留 base.tsx）

```bash
# 删除认证/验证码/特定网站脚本
rm contents/chat-auth-*.tsx contents/chat-captcha-*.tsx
rm contents/perplexity*.tsx contents/phind*.tsx contents/you*.tsx contents/devv_*.tsx
rm contents/search-engine-content.tsx
```

### Step 0.3: 删 component/ 旧组件

```bash
rm -rf component/xframe
rm component/common/ChatCaptchaBanner.tsx
rm component/ModelCheckList.tsx
```

### Step 0.4: 删 background/ 旧消息处理

```bash
rm -rf background/messages/copilot background/messages/kimi
rm background/messages/fix-partition-cookie.ts
rm background/messages/metaso-session.ts
rm -rf background/ports
```

### Step 0.5: 删其他

```bash
rm -rf resources/js rulesets/rule.json
rm utils/custom-fetch-for-chat.ts utils/UploadUtils.tsx utils/remove-console-in-porudciton.ts utils/request-log.ts
```

### Step 0.6: 简化 IBot / BotBase

删除 `startAuth()`, `startCaptcha()`, `uploadFile()`, `supportedUploadTypes` 及相关类型。

### Step 0.7: 简化 background/index.ts

删除 injectContentScript、GA4 相关逻辑，只保留 sidePanel open/close。

### Step 0.8: 简化 ModelManagementProvider

只保留 DeepSeekBot。

### Step 0.9: 简化 utils/index.ts

删除 `getCookiesInBackendByDomain`, `addCspParamsToUrl`, `addMobileHeaderToUrl`, `openInPlugin`, `btoaObj`, `atobObj`, `R_SCP_PARAM` 等旧常量。

### Step 0.10: 清理 package.json 无用依赖

```bash
pnpm remove ali-oss firebase websocket-as-promised lottie-web js-sha3 @ungap/custom-elements
```

### Step 0.11: 清理 .plasmo/messaging.d.ts

重新生成（删除旧消息类型声明），只保留需要的。

## 风险

1. **contents/base.tsx 依赖链深** — 需要仔细解耦
2. **conversation 页面改动大** — 可能影响 UI 渲染
3. **文件上传需重建** — 旧方案走 AI 网站的上传 API，新方案需本地解析后作为 prompt 发送
4. **构建可能反复失败** — 大范围删除后需多轮 tsc 修复
